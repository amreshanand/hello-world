Basic HTML Elements:

<HTML> - It is the top level element in every HTML document.
<head> - It is the element containing title and references to CSS or script file.
<body> - It contains main body of the webpage.

More About HTML:

Common elements include headings, paragraphs, links, lists, and images. Elements can have attributes like class and id to help with styling and scripting, and modern HTML also includes semantic tags like <header>, <main>, and <footer> to improve structure and accessibility.

Note: HTML elements are not case sensitive.
